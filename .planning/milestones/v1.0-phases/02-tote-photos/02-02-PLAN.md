- Gallery grid: Map photos to thumbnail images using `/api/photos/${photo.id}/thumbnail?source=${source}`, click opens lightbox at that index

- Render PhotoDeleteConfirm inline (controlled by showDeleteConfirm state)
- If photos array is empty, render nothing (parent decides placeholder)
- Use 'use client' directive
